# Robotics Corner Website

A professional website for Robotics Corner showcasing expertise in automotive/AUTOSAR, autonomous driving, and robotics.

## Running Locally

Navigate to the project directory and use one of the following methods:

### Option 1: Python (Recommended)
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

### Option 2: Node.js
```bash
npx -y serve .
```

### Option 3: PHP
```bash
php -S localhost:8000
```
